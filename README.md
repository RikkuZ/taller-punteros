# Taller de Punteros y Matrices
*Estudiante:* Juan Sebastian Mejia Camargo

## Descripción
Este programa resuelve:
1. Gestión de matrices dinámicas y sumas por fila.
2. Aplicación de funciones (cuadrado, doble, inverso) usando punteros a función.
3. Transformaciones con lambdas y composición de operaciones (Bonus).

## Compilación
g++ src/main.cpp -o taller_punteros
